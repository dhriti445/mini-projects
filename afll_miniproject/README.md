# AFLL Mini Project

## Description
Syntax validation of a Java program using Context-Free Grammar (CFG) for various Constructs

## Concepts Used
- Context-Free Grammar
- Syntax Analysis
- Compiler Design

## Constructs
- Simple Data-type Declaration
- Class Definition
- Simple Object declaration
- Array Declaration
- Looping Constructs
(can be tested for more constructs)

## How to Run
1. Compile the Java files
2. Run the main program(main.py)
